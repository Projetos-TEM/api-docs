API Prestadores

API REST para consulta de prestadores conveniados à Rede TEM.

URL ¶

A URL base depende do ambiente:

Homologação
https://qa-api-app.temrede.com.br
Produção
https://api-app.temrede.com.br

O endereço de produção só funciona para IP previamente liberado.

Autenticação ¶

Essa API exige autenticação usando o método OAuth 2.0. As credenciais para obter o token devem ser solicitadas previamente.
