API SAC

API REST para o Sistema de Atendimento ao Consumidor (SAC) da Rede TEM.

URL ¶

A URL base depende do ambiente:

Produção
https://api-sac.temrede.com.br

O endereço de produção só funciona para IP previamente liberado.

Autenticação ¶

Essa API exige autenticação usando o método OAuth 2.0.

Locais ¶

Obtém localizações onde há prestadores da Rede TEM.

Lista de Estados  ¶

POST/api/sac/states

Retorna todos os estados que há prestadores da TEM.

No retorno de sucesso, uma lista de estados é retornada.

Códigos de erro:

  • 400: para erros na requisição.

  • 401: para erros relativos à autenticação ou à autorização.

No retorno de sucesso, uma lista de eventos é retornada.

Códigos de erro:
